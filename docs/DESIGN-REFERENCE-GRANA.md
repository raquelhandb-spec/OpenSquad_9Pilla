# 🎨 Design Reference — App GRANA

**Data:** 14/06/2026  
**Fonte:** App de declaração de imposto "Grana"  
**Por que:** Minimalismo + Acessibilidade + Conversão

---

## 🎯 O Que Grana Faz Bem

### 1. **Paleta de Cores**
```
Verde Menta (Primária)   → #4ECDC4 ou similar
Branco Puro              → #FFFFFF  
Cinza Suave (Secundário) → #D4D4D8
Fundo (Claro)            → #F0F9F8 (quase branco, toque de verde)
```

**Por que funciona:**
- Verde = confiança, crescimento, tranquilidade
- Menta (vs. escuro) = acolhedor, moderno
- Branco puro = ar, respiro, clareza
- Cinza = textos secundários, não compete com CTA

---

### 2. **Tipografia**
```
Títulos Grandes:  Sans-serif clean, peso 600-700
                  "Declaração segura em 5 minutos"
                  (direto, promete valor)

Body Text:        Sans-serif limpo, peso 400
                  "Preencha seu email e comece..."
                  (microcopy clara, convidativa)

CTA:              Botão grande, texto simples
                  "Continuar" (ação clara)
```

**Por que funciona:**
- Sem serifas (moderno, tech)
- Peso variável (hierarquia óbvia)
- Sem decorações (minimalista = segurança)
- Microcopy conversa com usuário

---

### 3. **Ilustração 3D do Bonequinho**
```
✅ Pessoa REAL (não avatar genérico)
✅ Pose RELAXADA (não stressado)
✅ Cores CONSISTENTES com marca
✅ Minimalista (sem sobrecarregar)
✅ Humanidade (CTA secundário: "Já tem conta? Entrar")
```

**Por que funciona:**
- Reduz fricção (mostra que é fácil)
- Humaniza (não é robô/máquina)
- Cor verde reforça marca
- 3D moderno (não flat design genérico)

---

### 4. **Layout & Hierarquia**
```
[Esquerda: Bonequinho grande]
[Direita: Formulário minimalista]

⬇️

Título Grande (Promessa de valor)
    ↓
Microcopy (O que fazer)
    ↓
Campo de Email (1 input só)
    ↓
CTA Primário (Botão grande)
    ↓
CTA Secundário (Link "Já tem conta?")
```

**Por que funciona:**
- Split-screen visual (bonequinho + form)
- Fluxo claro (não confunde usuário)
- 1 input = não assusta
- Botão desabilitado até email válido (elegância)
- Link secundário em cinza (não compete)

---

### 5. **Microcopy (Textos Pequenos)**
```
"Preencha seu email e comece a usar o Grana"
"Preencha um e-mail válido para continuar."
"Já tem uma conta? Entrar"

⬇️ Características:
✅ Conversacional (não robótico)
✅ Claro (sem jargão)
✅ Ação esperada (verbo no início)
✅ Validação suave (não "ERRO!")
```
