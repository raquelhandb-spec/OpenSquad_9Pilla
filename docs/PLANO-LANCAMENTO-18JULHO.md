```
WORKSHOP R$49,90  →  porta de entrada

EBOOK R$29,90     →  oferta dentro do workshop (isca)

AULA INDIVIDUAL R$500  →  próximo passo natural pós-ebook
```

### Direção de arte para as peças

- **Paleta:** MOSS + BEGE + TINTA. Sem terracota (paleta anterior aposentada).
- **Tipografia:** Playfair Display nos títulos, DM Sans no corpo.
- **Tom visual:** sofisticado mas acessível. Mercado financeiro para pessoas reais, não para Wall Street.
- **Evitar:** linguagem de "ganho fácil", imagens de gráficos subindo infinitamente, linguagem de apostas.
- **Buscar:** clareza, confiança, a Raquel como guia de jornada, não guru.

### Palavras banidas em qualquer peça da 9Pilla

- ❌ Aposta / apostar / apostei (qualquer variação)
- ❌ Trader (sempre: operadora ou operadora de opções)
- ❌ Aprender no hook
- ❌ Travessão (—) em texto corrido
