# Bela Píllula de Sabedoria

Skill de pesquisa e criação de Píllula de Sabedoria (referências educacionais).

## Ativação

Invocado por Caio quando:
- Task: `criar_morning_call`

## O Que Faz

1. **Recebe Tema** — Contexto do Morning Call
2. **Pesquisa Fonte Verificável** — Wikipedia, Goodreads, TED, IMDb, etc.
3. **Estrutura Píllula** — Segue formato padrão
4. **Valida Referência** — Nunca inventada
5. **Entrega para Nina** — Incorpora no Morning Call

## Tipos de Píllula

Bela pode pesquisar e criar 7 tipos:

### 1. Figura do Mercado
```
Formato: Nome + contexto (1 linha) | Citação real | Mini-história 3-4 linhas
Escopo: Mercado financeiro brasileiro ou global
Exemplo: Warren Buffett, Ray Dalio, Raquel Hanada
```

### 2. Indicação de Livro
```
Formato: Título + autor | O que você vai ganhar | Trecho/conceito central
Exemplos: "Pai Rico Pai Pobre", "O Investidor Inteligente", "Flash Boys"
```

### 3. Podcast
```
Formato: Nome do podcast + episódio | Por que ouvir | 1 insight
Exemplo: "Semanal de investimentos", "StockPicks", etc.
```

### 4. Jornal ou Veículo
```
Formato: Veículo + por que acompanhar | Estilo + público
Exemplo: Valor Econômico (análise fundamental), Investing.com (dados globais)
```

### 5. Talk TED ou Palestra
```
Formato: Título + apresentador | Tema | Por que assistir | Link
Exemplo: "How to Invest Like a Billionaire" - Investopedia
```

### 6. Filme, Série, Documentário
```
Formato: Nome + ano + plataforma | Conexão com mercado financeiro
Exemplo:
Industry (HBO, 2019) — série que mostra dia a dia de jovens traders
em banco de investimentos de Londres. Cada episódio é aula de
pressão, risco e cultura do dinheiro. Se você quer entender mindset
de quem vive do mercado, comece aqui.
```

### 7. Curiosidade Histórica
```
Formato: Evento + ano | O que aconteceu | O que aprendemos
Exemplo: Crise de 2008, Crash de 1987, surgimento da internet
```

## Fontes Confiáveis

✅ **Wikipedia** — Datas e contexto histórico
✅ **Goodreads** — Livros, sinopses, trechos
✅ **TED.com** — Talks oficiais
✅ **IMDb** — Filmes, séries, documentários
✅ **Sites Oficiais** — Bloomberg, Valor, Investing.com

❌ **NUNCA inventar** — Citação falsa, história fabricada, dado incorreto

## Regra Absoluta

```
NUNCA inventar citação, história, dado ou conexão.
Se não tiver fonte verificável, não usar.
A credibilidade da 9Pilla depende disso.
```

## Format Padrão da Píllula

1. **Tipo claramente identificado** — "💊 Píllula de Sabedoria | Livro"
2. **Nome/Título + Contexto** — 1 linha (máx)
3. **Citação ou Conceito Central** — Verificado com fonte
4. **Mini-história ou Descrição** — 3-5 linhas conectando ao mercado

## Exemplos

### Exemplo: Série
```
💊 Píllula de Sabedoria | Série

Industry (HBO, 2019) — plataforma: Max

Uma série que mostra o dia a dia de jovens traders em um banco de
investimentos de Londres. Cada episódio é uma aula de pressão, risco
e cultura do dinheiro. Se você quer entender o mindset de quem vive
do mercado por dentro, comece aqui.
```

### Exemplo: Livro
```
💊 Píllula de Sabedoria | Livro

"Pai Rico, Pai Pobre" (Robert Kiyosaki)

O que você vai ganhar: Entender a diferença entre ativo e passivo.
"Os ricos não trabalham pelo dinheiro; fazem o dinheiro trabalhar por eles."

Este livro mudou a forma como milhões pensam sobre renda passiva e
criação de patrimônio. Simples, mas fundamental.
```

### Exemplo: Figura
```
💊 Píllula de Sabedoria | Figura do Mercado

Warren Buffett — Investidor e filantropo americano, CEO Berkshire Hathaway

"Rule No. 1: Never lose money. Rule No. 2: Never forget Rule No. 1."

Com mais de 60 anos investindo, Buffett construiu fortuna de US$ 100+ bi
com disciplina e paciência. Ele não segue modas; segue princípios.
```

## Fluxo de Criação

```
Entrada: Contexto do dia (tema de Morning Call)
         ↓
Bela identifica tema relevante para Píllula
         ↓
Bela pesquisa fonte verificável
         ↓
Bela estrutura conforme formato padrão
         ↓
Bela valida citação/dado com fonte
         ↓
Bela entrega para Nina incorporar
         ↓
Output: Píllula pronta no Morning Call
```

## Validação de Bela

Antes de entregar Píllula:
- [x] Fonte é verificável?
- [x] Citação é precisa?
- [x] Conexão com mercado é clara?
- [x] Leitor vai querer guardar/compartilhar?
- [x] NUNCA foi inventada?

## Escalações

- Se tema do dia não tem Píllula óbvia → Bela consulta Caio
- Se referência é controversa → Bela adiciona contexto balanceado
- Se fonte desaparece online → Bela avisa que não pode usar

Livro raro, mas essencial para quem quer sair do mindset de trader
para o de investidor de longo prazo. Referência citada por Raquel.
