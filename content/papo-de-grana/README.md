```

├── ep-01/
│   ├── ep-01.md (conteúdo principal)

```

## Padrão de conteúdo

- **Tamanho:** 25.000–35.000 palavras
- **Estrutura:** Abertura (hook) → Corpo (educação) → Encerramento (reflexão + CTA)
- **Tom:** Raquel conversando com uma amiga
- **Inclui:** Neurociência, economia comportamental, dados com fontes, simulações matemáticas

## Elementos obrigatórios

✅ **Abertura aprovada:**  
> "Existe um momento na vida em que o barulho externo silencia…"

✅ **Tagline (fixo):**  
> "Liberdade não se aposenta. Se constrói todo dia."
