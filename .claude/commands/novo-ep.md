# /novo-ep

Inicializa a estrutura de um novo episódio da série **Papo de Grana**.

## Estrutura

A série tem **9 episódios** (Temp. 1), cada um com:

- **Título** (aprox. 50 caracteres)
- **Abertura** (hook narrativo forte)
- **Corpo principal** (25.000–35.000 palavras)
  - Neurociência + economia comportamental
  - Dados com fontes verificáveis
  - Simulações matemáticas
  - Exemplos práticos
- **Encerramento** (reflexão + chamada para ação)
- **Tagline** (fixo): *"Liberdade não se aposenta. Se constrói todo dia."*

## Tom

Raquel conversando com uma amiga. Acessível, sem julgamento, sem economês, empoderador.

## Comando

```
/novo-ep 02
```

Saída:
- Pasta criada: `content/papo-de-grana/ep-02/`
- Arquivos:
  - `ep-02.md` (template vazio)
  - `fontes.md` (referências)
  - `draft-01.md` (primeira versão)
  - `review.md` (checklist)

## Checklist de qualidade

- [ ] 25k–35k palavras
- [ ] Mínimo 3 fontes por seção
- [ ] Nenhuma promessa guru ou enriquecimento rápido
- [ ] Linguagem acessível (Flesch-Kincaid Grade 6–7)
- [ ] Tagline preservada
- [ ] Português brasileiro
- [ ] Reviso de fatos/dados
- [ ] Tom Raquel-conversando-com-amiga
